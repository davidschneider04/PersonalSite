Auto-format a Resume.

The purpose of this project was to create the simplest way possible to edit and share my resume while still maintaining custom formatting. I first created custom styles in Word, saved those as a template, then use Python to apply these styles across a formatted JSON to create a stylized .docx file. Then, I created a very hacky automation using PyAutoGUI to open Word and save the the .docx is converted as .pdf, since it seems like this option preserved the formatting better than anything I could figure out with command line. The .pdf is then converted to an .svg for use in mobile iframes. Finally, both are uploaded to GitHub and AWS to update my personal page and other sites I represent myself on. Special thanks to: PDF2SVG and python-docx
